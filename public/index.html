📋 Import from Browser

Lowe's blocks automated scans, so grab deals from your own browser:
1. Drag this button to your browser's bookmark bar (one-time):
📋 Grab Lowe's Deals
2. On lowes.com, open a Deals listing for your store, then click the bookmarklet. It pages through the whole category automatically (watch the counter, top-right) and copies everything in one go.
3. Paste below and Import.

Home Depot has a real product API — this grabs the whole clearance feed with exact discounts:
1. Drag this to your bookmark bar (one-time):
📋 Grab Home Depot Deals
2. On homedepot.com, open a clearance / Special Values listing for your store, click the bookmarklet (it "arms"), then click page 2 / Next (or change Sort) once — it pages the whole feed automatically (watch the counter, top-right).
3. Paste below and Import.

⚙️ Automated scanning (beta)

This checks whether the server can pull Home Depot deals on its own (for scheduled scanning). It does not change the manual grab above. Three steps, all on a HD Special Values listing page:

1. Cookies — install the free Cookie-Editor extension, open the listing page, click Cookie-Editor → Export → JSON, paste into the first box.
2. API request — drag this to your bookmarks bar, then on the listing page click it and click page 2 / Next once; it copies the exact request → paste into the second box: 📋 Capture HD API request
3. Click Test server access (do it within a few minutes — cookies expire fast).
